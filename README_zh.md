## Quantum Graphics Toolkit
Quantum Graphics Toolkit (以下简称 QGT) 是一个基于OpenGL封装出的图形库<br>
它保留了OpenGL极致灵活的特性，可以很简单也可以很复杂，你可以用寥寥几行代码写出一个GUI应用，也甚至可以用稍稍多一些的代码，做出一个好玩的2D游戏
